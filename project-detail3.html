Image Enhancement on FPGA using Verilog

Plateform

Team

chaitanya Patel
keval Patel

Technologies

Software

Xilinx ISE, MATLAB, Verilog

Date

Digital image processing plays a vital role in the analysis and interpretation of remotely sensed data. Especially data obtained from Medical and Satellite Remote Sensing, which is in the digital form, can best be utilized with the help of digital image processing. Thus, in this project we did image processing using verilog. Moreover, Fpga is highly alternative of fast performance of digital signal processing application. These algorithms are implemented in Verilog HDL using Xilinx ISE, MATLAB.

Firstly, we converted image file in .bmp extension beacause, in Verilog we need .hex file. so, we converted .bmp file to .hex file using Matlab software. Then after we processed .hex file in verilog software with verious image processing for emaple inverting, thresholding, and brightening operations. we got .bmp files as a output. the aim and motive of this project was Verilog system is much faster and hardware system compare to other system.
